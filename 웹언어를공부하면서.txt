웹 개발을 한 번 배워보고 싶어서 프로젝트를 진행하게 되었다

웹 개발 = 웹사이트를 개발하는 모든 개발
웹 개발의 종류 = 프론트엔드, 백엔드, 퍼블리싱
    - 백엔드는 눈에 보이지 않는 것 구현 (데이터베이스, 서버 등)
    - 프론트엔드는 눈에 보이는 것 구현 (디자인 + 기능)
    - 퍼블리싱은 디자인만 구현
퍼블리싱을 하기 위해 html, css, javascript 필요

사실 html, css, 간단한 javascript를 배우는 것도 어려웠지만
무엇보다 어려웠던 건 github으로 호스팅 하는 거였다
C를 기반으로 python과 java 등의 언어를 한 번씩 해봐서 그런가
web 언어를 이해하는 것은 어렵지 않았고,
처음에 "<div> 내용 <div \>" 등의 표현이 익숙치 않았는데
눈에 익으면서 오히려 C나 다른 프로그래밍 언어보다 쉽다는 인상을 받았다.

html을 이용하여 내용을 작성하는 것은 어렵지 않았는데,
CSS로 웹 페이지를 예쁘게 꾸미는 것이 특히나 어려웠다.
왜 웹 디자이너들을 뽑는지 전에는 몰랐는데 웹 페이지를 간단하게나마 만들어보니
확실히 디자인쪽 센스가 있어야한다는 것을 알았다.
색감을 이용해 페이지를 예쁘게 만드는 것은 둘 째 치고
페이지 레이아웃(컬럼 나누기 등)을 예쁘게 만드는 부분이 특히나 어려웠다.
하지만 한 번 이해하고, 실제로 만들어보면서 체화를 하고 나니
지금 보면 처음 배울 때 왜 어려워했는지 모르겠는 부분이다.
